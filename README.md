# We the Users — the open drafting repo

**A call to draft a Declaration of Internet Independence.** Launched **July 2, 2026** —
the 250th anniversary of the day the Continental Congress voted for American independence.
The site: **https://wetheusers.ai**

This repository is the drafting table. Every document here is a working draft —
**discussions and pull requests are welcome on any piece of this work**: a word, a clause,
a whole argument.

## Repository layout

- `docs/declaration/` — the provisional Declaration draft.
- `docs/essay/` — the six essays making the public case, with their PDFs.
- `docs/founding-brief.md` (+ `.pdf`) — the founding brief: the full vision, the
  architecture, the hard questions, and the open decisions. Candid by design. **Start here.**
- `docs/constitution/` — where the Covenant and the Code will be drafted (later phase).
- `docs/outreach/` — the outreach packet.
- `references/` — grounding research notes.
- `source-notes/` — the founder's early typed working notes, preserved as provenance.
- `server/` — the open backend behind "Join the call" (a Cloudflare Worker with a D1
  database): schema, seed communities, and worker source. Nothing is run in secret.
- `deck/` — the launch deck and its build script.
- `build/pdf/` — the document build pipeline: PDF generation and the version registry.
- `CHANGELOG.md` / `VERSIONS.md` — the narrative history and current document versions.

## How to take up the pen

- **Discussions** — argue with any idea, or propose a new one.
- **Pull requests** — edit any document directly. The smallest useful edit is welcome.
- **The site** — read everything in one place, and add your name: https://wetheusers.ai

## How these drafts were made

With Claude Code, to get the ideas onto the page fast. Speed was the point — so is the
humility. None of this is a final product; the final version is the one written together.
